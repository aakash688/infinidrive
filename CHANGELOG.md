# Changelog

All notable changes to InfiniDrive will be documented in this file.

## [Unreleased]

### Added
- API & Projects system for programmatic access
- RESTful API v1 endpoints
- Project-based file organization
- API key management with permissions
- Folder management via API
- Upload speed and ETA display
- Parallel chunk uploads for faster uploads
- Search functionality in file manager
- File renaming and moving

- Improved upload performance with parallel chunk uploads
- Enhanced file manager UI/UX

- File download/stream authentication
