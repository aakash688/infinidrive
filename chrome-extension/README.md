# InfiniDrive Chrome Extension

Chrome extension for quick file uploads to InfiniDrive.

## Setup

1. Build TypeScript:
```bash
cd chrome-extension
npm install
npm run build
```

2. Load in Chrome:
- Open `chrome://extensions/`
- Enable "Developer mode"
- Click "Load unpacked"
- Select the `chrome-extension` folder
