# InfiniDrive Web Panel

SolidJS web application for InfiniDrive cloud storage.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Set API URL in `.env`:
```
VITE_API_URL=http://localhost:8787
```

3. Run dev server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Deploy to Cloudflare Pages

The `dist/` folder can be deployed to Cloudflare Pages.
